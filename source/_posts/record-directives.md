# 隨手記一點常用 Hexo 指令

``` bash
# Server: hexo server
$ hs
```

``` bash
# Clean: hexo clean
$ hc
```

``` bash
# Generate: hexo generate
$ hg
```

``` bash
# Deploy: hexo deploy
$ hd
```

``` bash
# New Blog: hexo new <blog_title>
$ hexo new <blog_title>
```

``` bash
# New Page: hexo new page <page_name>
$ hexo new page <page_name>
```
